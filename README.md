# 坦克大战 (Tank Battle)

纯 HTML5 Canvas 坦克大战游戏，单文件，浏览器打开即玩。

> 本项目由 AI 全量生成，使用 **DeepSeek V4 Pro** API 驱动实现，包括游戏逻辑、渲染、音效、关卡设计及本 README。Claude Code 作为命令行交互工具执行操作。

## 快速开始

直接用浏览器打开 `index.html` 即可。

## 操作

| 按键 | 功能 |
|------|------|
| W A S D | 移动 |
| 鼠标移动 | 瞄准 |
| 鼠标左键 / 空格 | 射击 |
| R | 死亡后重新开始 |

## 代码框架

```
index.html
├── CSS          # 暗色主题 UI 样式，绿光 CRT 风格
├── HTML         # Canvas 画布 + HUD（得分/生命/关卡）+ 操作提示
└── JavaScript   # 全部游戏逻辑（~800 行）
    ├── 音频系统     playSound() — Web Audio API 合成音效
    ├── Tank 类    玩家 & 敌人坦克，含移动、碰撞、AI、渲染
    ├── Bullet 类  子弹，含飞行、墙体碰撞、渲染
    ├── Wall 类    墙壁（不可破坏 / 可破坏），含生命值
    ├── Particle 类 爆炸粒子特效
    ├── PowerUp 类  道具（回血、加速、速射），带脉冲动画
    ├── 关卡系统    预置 3 张地图 + 第 4 关起程序化随机生成
    ├── 游戏循环    gameLoop() — 60fps requestAnimationFrame 主循环
    └── 输入处理    键盘 + 鼠标事件
```

### 核心设计

- **单文件架构** — CSS / HTML / JS 全在一个文件中，零依赖
- **Canvas 渲染** — 所有图形通过 2D Context API 绘制（坦克、炮塔、履带、子弹、粒子）
- **Web Audio 音效** — 用振荡器实时合成射击/爆炸/道具/死亡音效，无外部音频文件
- **AI 行为** — 敌人混合追击玩家 + 随机漫游，撞墙时自动转向
- **碰撞检测** — AABB 包围盒，子弹命中、坦克互撞、墙壁阻挡
- **关卡伸缩** — 前 3 关手绘地图，4 关起用伪随机数种子生成无限变体
- **难度曲线** — 随关卡提升：敌人数量增加、速度加快、出现快速/重型敌人

## 敌人类型

| 类型 | 颜色 | 特点 |
|------|------|------|
| 普通 | 红色 | 标准速度，1 血，100 分 |
| 快速 | 橙色 | 高移动速度、高射速，100×3 分 |
| 重型 | 紫色 | 慢速、3 血、体积更大，100×2 分 |

## 道具

| 道具 | 符号 | 效果 |
|------|------|------|
| 回血 | + | 恢复 1 点生命（上限 5） |
| 加速 | » | 移动速度提升，持续 5 秒 |
| 速射 | 闪电 | 射击冷却大幅缩短，持续 4 秒 |

## 技术栈

- HTML5 Canvas
- Web Audio API
- Vanilla JavaScript (ES6+)
- 无框架、无构建工具、无外部依赖
